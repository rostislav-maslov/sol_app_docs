# Sol.app

Этот репозиторий нужен для хранения общей информации по проекту `Sol.app`.

## Назначение репозитория

- хранить общую документацию и контекст по проекту;
- фиксировать задачи, эпики и договоренности;
- использовать репозиторий как точку постановки и уточнения работ.

## Связь с рабочими репозиториями

Задачи и решения формируются в этом репозитории, а затем реализуются в
профильных репозиториях проекта:

- backend-репозитории;
- Flutter-репозитории.

## Роль этого репозитория

Этот репозиторий выступает общим управляющим и документационным контуром для
проекта `Sol.app`, где команда может:

- собирать общую информацию по продукту и проекту;
- описывать задачи перед реализацией;
- согласовывать контекст перед переносом работы в технические репозитории.
