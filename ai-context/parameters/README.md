# AI Context Parameters

Эта директория задает стандарт хранения параметров использования `ai-context`.

Параметры разделяются на два scope:

- `repository/` - versioned параметры конкретного репозитория;
- `local-machine/` - локальные параметры конкретной машины, которые не должны
  коммититься в git.

## Зачем это нужно

`ai-context` используется не только как набор правил, но и как рабочий слой
проекта. Поэтому часть параметров должна быть общей для репозитория, а часть
остается локальной и чувствительной.

## Базовый принцип

- Repository-level параметры можно коммитить.
- Local-machine-level параметры коммитить нельзя.
- Для обоих scope в baseline должны быть README и шаблоны.
- Реальные локальные секреты создаются пользователем локально и игнорируются
  git.
