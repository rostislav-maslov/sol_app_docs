# Local Machine Parameters

Эта директория хранит local-machine-level параметры использования `ai-context`.

Такие параметры:

- зависят от конкретной локальной машины;
- могут содержать секреты;
- не должны коммититься в git.

## Что сюда можно класть

- логины и пароли для Bitrix;
- локальные токены и API keys;
- machine-specific доступы к Jira, Bitrix, другим системам;
- любые пользовательские секреты и credentials.

## Правило

- Реальные локальные параметры создаются здесь локально пользователем.
- В git можно хранить только `README.md`, `.gitignore` и example-файлы.
- Любой файл с реальными значениями должен оставаться неотслеживаемым.

## Рекомендуемый старт

Возьми за основу `local-machine.example.yaml` и создай свой локальный файл,
например:

- `local-machine.yaml`
- `bitrix.local.yaml`
- `integrations.local.yaml`

Все такие файлы будут игнорироваться git по правилам этой директории.
