# Скрипты `ai-context`

Эта директория хранит локальные утилиты, которые входят в baseline-процесс
работы AI-агента.

## Базовый набор

- `show-completion-alert.sh` - обязательный единый способ показать модальный
  алерт о завершении задачи на macOS.

## Принцип

- Если какой-то скрипт становится частью обязательного процесса, это должно
  быть явно отражено в `ai-context/ai-rules` или `ai-context/tasks/README.md`.
- Project-specific утилиты можно добавлять сюда отдельно, но они не должны
  подменять baseline-скрипты без необходимости.
