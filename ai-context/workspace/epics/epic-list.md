# Epic List

Короткий рабочий список эпиков команды для режима `project-manager`.

## Рабочий формат

Каждый эпик в секции `Epics` хранится одной строкой:

```text
🔴 [EP-001] Короткое название эпика
```

Где:

- `🔴` - статус эпика;
- `EP-001` - стабильный код эпика;
- подробная постановка и декомпозиция хранятся в `./<код>/`.
